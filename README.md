# Aether — The Universal AI Agent Framework

> 博采众长，无所不达

Aether is a next-generation AI agent framework combining the best features from 20+ global AI agent products. It works natively on **Windows 11** and **Linux**.

## Quick Start

```bash
# Install
pip install aether-agent

# Run
aether
```

## Features

- **Cross-platform**: Native Windows 11 + Linux, no WSL required
- **Protocol-first**: MCP tools + A2A agent collaboration
- **Memory system**: Persistent cross-session memory with conflict resolution
- **Skills system**: Reusable procedural knowledge with community marketplace
- **Multi-agent orchestration**: Sequential, parallel, hierarchical, conversational
- **Security**: 6-level permission system + Docker sandbox + circuit breaker
- **Cron engine**: Scheduled LLM-driven tasks and script jobs

## Development

```bash
# Clone
git clone https://github.com/aether-agent/aether.git
cd aether

# Install dev dependencies
pip install -e ".[dev]"

# Run tests
pytest
```
